# 小说自动生成器 - Grok版

一个基于Grok API的AI小说自动生成工具，支持GUI界面、用户确认机制和断点续传功能。

## 功能特性

### 核心功能
- **四层递进式生成**：设定圣经 → 章节大纲 → 场景分解 → 批量写作
- **用户确认机制**：每层生成后可修改或重新生成，5秒倒计时自动确认
- **断点续传**：支持中断后从任意阶段继续生成
- **三轮迭代写作**：初稿 → 扩充 → 润色，确保字数和质量

### 技术特点
- **智能分批处理**：根据目标字数动态调整批次大小
- **动态参数调整**：温度、max_tokens等根据生成阶段自动调整
- **限流处理**：自动处理API限流和重试
- **后端监控**：检测服务不可用并优雅退出

### GUI功能
- API配置管理（Base URL、API Key、Model选择）
- 实时日志输出
- 进度条显示
- 模型列表获取
- 连接测试
- 日志保存

## 系统要求

- Python 3.8+
- tkinter（通常随Python一起安装）
- 依赖包：`openai`

## 安装

```bash
pip install openai
```

## 使用方法

### 启动应用
```bash
python no.py
```

### 基本流程

1. **配置API**
   - 填写Base URL（默认：https://api.x.ai/v1）
   - 输入API Key
   - 选择模型（可点击"获取模型列表"自动获取）
   - 点击"测试连接"验证

2. **设置小说参数**
   - 输入主题（一句话创意）
   - 输入要求（字数、风格、特殊需求等）
   - 可选：指定章节数和每章字数

3. **开始生成**
   - 点击"开始生成"
   - 依次确认各阶段的生成内容
   - 可修改内容或选择重新生成

4. **断点续传**
   - 生成中断后，点击"继续上次生成"
   - 从中断处继续生成

## 生成流程详解

### 第1层：设定圣经
生成小说的核心设定，包括：
- 核心创意解读
- 世界观设定
- 角色档案
- 完整情节大纲
- 风格锁定

### 第2层：章节大纲
基于设定圣经生成各章节的详细大纲，包括：
- 核心事件
- 冲突点
- 情绪基调
- 角色出场
- 结尾钩子

### 第3层：场景分解
将每章拆解成具体的写作场景，包括：
- 场景摘要
- 关键角色
- 情绪标签
- 具体地点

### 第4层：批量写作
三轮迭代写作：
1. **初稿**：按场景安排写出初稿
2. **扩充**：扩充环境描写、角色内心、对话等
3. **润色**：检查连贯性和文字质量

## 配置文件

### novel_generator_config.json
保存API配置和章节设定：
```json
{
  "api_key": "your-api-key",
  "base_url": "https://api.x.ai/v1",
  "model": "grok-4-1-fast-non-reasoning",
  "chapters_count": "10",
  "words_per_chapter": "1200"
}
```

### novel_resume_state.json
保存断点续传状态（自动生成）

## 输出文件

生成完成后，小说保存为：`novel_YYYYMMDD_HHMMSS.txt`

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| target_words | 目标总字数 | 50000 |
| words_per_chapter | 每章字数 | 1200 |
| words_per_scene | 每场景字数 | 280 |
| chunk_size | 每批章数 | 动态计算 |
| ROUNDS | 迭代轮数 | 3 |

## 常见问题

### Q: 如何获取Grok API Key？
A: 访问 https://console.x.ai/ 注册并获取API Key

### Q: 生成速度慢怎么办？
A: 
- 减少目标字数
- 减少章节数
- 检查网络连接
- 查看是否触发限流

### Q: 生成中断如何恢复？
A: 点击"继续上次生成"按钮，从中断处继续

### Q: 如何修改生成的内容？
A: 在确认对话框中直接编辑文本，或选择"重新生成"

## 技术架构

```
NovelGenerator (核心生成引擎)
├── layer1_setting_bible() - 生成设定
├── layer2_chapter_outlines() - 生成大纲
├── layer3_scene_breakdown() - 场景分解
└── layer4_write_scenes() - 批量写作

NovelGeneratorGUI (GUI界面)
├── _setup_ui() - 界面初始化
├── _start_generation() - 启动生成
├── _show_confirm_dialog() - 确认对话框
└── 各种回调函数
```
